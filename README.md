# 🔐 Hash

**Hash** is a **Decentralized Identity & Authentication System** that eliminates passwords and enables secure login using blockchain and cryptography.

---

## 🚀 Overview

Hash allows users to authenticate using their crypto wallet instead of relying on centralized identity providers. It uses decentralized identifiers (DIDs) and cryptographic signatures to ensure secure and user-controlled authentication.

---

## 🔐 Key Features

* 🔑 **Passwordless Authentication**
  Login using wallet signatures

* 🪪 **Decentralized Identity (DID)**
  Users fully control their identity

* 🔄 **Secure Session Establishment**
  Uses Diffie–Hellman key exchange

* 🔒 **Encrypted Communication**
  Ensures secure data transfer

---

## 🧠 How It Works

1. User connects wallet (e.g., MetaMask)
2. Server sends a random challenge
3. User signs the challenge using private key
4. Server verifies using public key
5. Diffie–Hellman key exchange generates shared key
6. Communication is encrypted
